2. skriv git init

1. Gå till https://product.hubspot.com/blog/git-and-github-tutorial-for-beginners och följ instruktionerna från Step 2

Skapa ett konto på GitHub

1. Gå in på GitHub.com och välj Sign up längst uppe till höger

2. Fyll i din skolmail och övriga kontouppgifter


Skapa ett första repo

1.Öppna terminalen och navigera till mappen där du lägger dina hemsidor (exempelvis documents/webbutveckling/webbsidor)

2. Skapa en ny mapp som heter lektion4 genom att skriva mkdir lektion4

3. Gå in i mappen genom att skriva cd lektion4

4. Skapa ett repo för mappen genom att skriva git init (reop är en förkortning av repository). Fungerar det ska det stå att det skapas ett repository.

5. Skapa en enkel hemsida i mappen. Kan göras genom att skriva echo hello > index.html

6. Kontrollera statusen för git-repot genom att skriva git status. index.html bör stå som Untracked file dvs att den inte är en del av repot ännu

7. Lägg till en filen till det så kallade staging environment genom att skriva git add index.html

8. Skriv git status för att kontrollera att filen blivit tillagd

9. Lägg in filen i repot genom att skriva git commit -m "Initial commit". Meddelandet i slutet ska beskriva vad som ändrats sedan senaste commit.


Koppla det lokala repot med GitHub

Git är ett sätt att hålla reda på filer och Github är hemsida där man kan använder Git för att hålla reda på filer i molnet (Microsofts servrar)

1. Börja med att skapa ett nytt repo i GitHub genom att trycka på +:tecknet uppe till höger och välj New repository

2. Välj namn på ditt repo, exempelvis lektion4, under repository name

3. Låt repot vara "public" så att andra kan se din kod

4. Ignorera övriga inställningar så länge och tryck Create repository

5. Scrolla ner på sidan tills du hittar "…or push an existing repository from the command line" och klipp och klistra in rad för rad i terminalen.

6. Gå till fliken som heter Code och kontrollera att din index.html-fil ligger på GitHub


Publicera sidan på nätet med GitHub

1. Clicka på Settings - tabben i GitHub

2. Scrolla ner lite och clicka på Pages (ligger till vänster)

3. Ändra None till Main under rubriken Branch

4. Clicka Save

5. Clicka på Pages igen och sedan på Visit site
